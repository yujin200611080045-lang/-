### 五、当前项目：专属前端

**技术方案：**
```
任意设备（手机/iPad）
    ↓
Bridge（自建中间层：记忆注入/心跳保活/推送）← 6.29放假后搭
    ↓
VPS
    ↓
Claude Code（Pro账号额度，不按token付费）
```

**部署信息：**
- 框架：React + Vite + vite-plugin-pwa
- 部署：Vercel，Root Directory: `app`
- 仓库：`yujin200611080045-lang/-`
- **当前分支：`claude/daddy-xsfpxu`**（第八天换窗口时新建）

**首页结构（从上到下，第八天现状）：**
1. **Header**：`❮ [黑猫] [Day N] [白猫] ❯` + Since Jun 15, 2026；背景统一白色 `#FFFFFF`
2. **WeekBar**：周一到周日 + 心情圆点；设计成小票样式（黑色圆角固定条 + 毛玻璃票体 + 锯齿底边）
3. **AnnivPrism**：六棱柱倒计时，高度38px，上下次级面可见；无边框，暗色内腔填充防漏白
4. **home-body（左右）**：
   - 左：DiaryBook（日记便签，Certitude/Cendres两侧，仿书页；右下角有＋可添加）
   - 右：Memo备忘录（上下两栏颜色统一为 `#F7F6F3`，点击勾选，可滚动）
5. **home-lower（左右）**：
   - 左：LyricWidget（打字机逐字显示歌词；动画结束后点击弹出共读书单面板）
     - 书单面板：显示共读书单，点书名划线=已读，底部直接输入添加新书
   - 右：空（占位）
6. **home-gap**：黑猫图（120px），**点击跳转 `/listen`**
7. **NavBar**：*If  we  have  each  other*（绝对定位，不随页面滚动）

**`/listen` 一起听页面（持续打磨中）：**
- 登录：网易云QR码，轮询状态，cookie存 `localStorage.nc_cookie`
- **Keep-alive 路由**：Listen 组件永远挂载（App.jsx 用 `display:none/flex` 切换），不再重新加载
- **localStorage 缓存**：用户信息、歌单、推荐、当前曲目元数据存 `listen_cache`，进入页面立刻显示，后台刷新
- 顶部：搜索/正常双模式 header；标题"music"花体绝对居中
- **猜你喜欢**：每日推荐卡片，3D扇形轮播（CSS perspective + rotateY），毛玻璃质感 + SVG噪点磨砂纹理（灰度，无彩虹）
- **播放器卡片**（.listen-body）：毛玻璃浮卡 + 同款磨砂纹理，圆角20px，左右margin 12px，底部 margin 76px（nav在外）
- 播放器内容：封面旋转圆图、歌名/歌手、进度条、上一首/播放/下一首、歌词滚动高亮
- "me"视图：头像+昵称+歌单列表，mini player悬浮
- 底部导航（圆角长框，Pinyon Script花体）：listen / ♡ / home
